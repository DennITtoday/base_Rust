Дано целое неотрицательное число N, определите число десятков в нем (предпоследнюю цифру числа). Если предпоследней цифры нет, то можно считать, что число десятков равно нулю. Числа 0 < N < 10^19
You have a number N. You need to find the number of tens in it (the second last digit of the number). If there is no second last digit, you can consider that the number of tens is zero. Numbers 0 < N < 10^19

Try with this input: 
444

Sample output: 4
